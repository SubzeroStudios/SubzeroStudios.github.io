C Cheat Sheet For Dummies (heh) Basic Structure of The Project The structure of the project is setup between two functions: setup and loop. The setup function runs once when the program is started whereas the loop function runs over and over again until the board is turned off Example void setup() { // Code to run ONCE when the program is started } void loop() { // Code to run repeatidly after the setup function } Having a loop with no delay will cause the loop to run over and over again, thousands and thousands of times a second. This is extremely CPU intensive and unnessicary in our case. To add delay to our loop, and anywhere else in the code, we use the delay function Example // Syntax: delay(timeInMilisecondsToBeDelayed) void setup() { // Code to run ONCE when the program is started } void loop() { // Code to run once every second (1000 milisconds = 1 second) delay(1000); } Now we have a solid structure and we can now begin coding Variables And Such Unlike in Python, we must specify what type of variable we will be creating when we give it a name. There are many different kinds of variables, but here are the ones you will need to know int (short for integer) -> A whole number -> 0, 1, 2, etc double -> A decimal number -> 0.1563, 54.123, 69.1, etc Declaring a variable To set a variable in C, you must state the type, then the name, then an equal sign (provided you are giving that variable an initial value), then its value, then a semicolon Example int apples = 4; int oranges; // you do not have to give it an initial value if you do not wish to double money = 5.23; double percent = 0.96; If / else if / else statements If, else if, and else statements work the exact same in C as they do in Python; the only catch is the syntax is slightly different Example if(condition) { // code to be run if the condition was true } else if(condition) { // code to be run if the if statement's condition was false but this one's was true } else // notice there is no condition { //code to be run if none of the past if/else if statements' conditions were true } What are conditions though? Here are some examples: Example if(9 > 3) { // code that WILL run because 9 is greater than 3 } int sides = 4; if(sides <= 3) { // code that will NOT run because the variable 'sides' is not less than nor equal to 3 } Relational Operators Relational Operators are what let us compare variables and numbers to each other. They are as such: < greater than5 > 4 is TRUE < less than4 < 5 is TRUE >= greater than or equal4 >= 4 is TRUE <= less than or equal3 <= 4 is TRUE == equal to5 == 5 is TRUE != not equal to5 != 4 is TRUE Using The Pins of The Microcontroller Before we begin, it is useful to understand some basic lingo on microcontrollers. Pins on the microcontroller can be turned on and off. This can allow you to, for example, turn an LED on and off. When we turn pins on and off, it is typically called "setting them HIGH" or "setting them LOW" (HIGH = on, LOW = off). This is because the pin is never at any one point completely on or completely off; You can change it to its HIGH or LOW voltage. Don't worry about this however, just think of it as you are turing the pins on and off (it'll make a lot more sense) To set a pin on the Arduino to HIGH or LOW, we must first set it up as an output pin (you can also use it as an input pin but that's not needed for this project) Example // Syntax: pinMode(pinNumber, OUTPUT); pinMode(9, OUTPUT); // sets pin 9 as an output Now we get to turn the pin on and off (HIGH and LOW) Example // Syntax: digitalWrite(pinWeSetupAsOutput, LOW/HIGH); digitalWrite(9, LOW); // turns pin 9 off digitalWrite(9, HIGH); // turns pin 9 on Combining What We Have Learned So Far void setup() { pinMode(7, OUTPUT); digitalWrite(7, LOW); } void loop() { int numberOfFeetToGo = 6; if(numberOfFeerToGo <= 10) { digitalWrite(7, HIGH); } delay(1000); } This simple program checks every second to see if our variable, 'numberOfFeetToGo', is less than 10. If it is, it turns pin number 7 (which is off from the start) on. For Loops For loops are used when you want to execute a certain block of code a certain number of times. Example for (int i = 0; i < 8; i++) { // code to be executed 8 times } The variable 'i' is a counter variable; it starts off at 0 (as shown by the 'int i = 0') and increases by 1 (as shown by the 'i++') every time it loops. Every loop, it checks to make sure that the middle condition ('i < 8') is true. If not, the loop stops. You can change the 'i' to whatever you want, but generally as a convention it is kept as i (unless you have a for loop inside of a for loop which is called a 'Nested For Loop'). You can replace the 8 with a variable (or an expression) Example int apples = 12; int oranges = 4; for (int i = 0; i < (apples+oranges); i++) { // code to be executed 16 times } Functions Functions are chunks of code that you can run when you wish by calling on them. They are syntactically made up of a return type, a name, and parameters. I won't bother teaching return types as you won't need to know them for this project. Simply use 'void' to indicate there is no returned variable.
